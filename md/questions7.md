## 1. No Show ครบ 5 ครั้ง — เกิดอะไรกับบัญชี?

ตอบแล้วว่า "สะสม 5 ครั้ง มีการแจ้งเตือนบัญชี"  
แต่ยังไม่รู้ว่าหลังแจ้งเตือนแล้วเกิดอะไรต่อ

**Q:** เมื่อผู้ใช้สะสม No Show ครบ 5 ครั้ง ระบบทำอะไร?
- [ ] แจ้งเตือนอย่างเดียว (Notification) ไม่มีผลอื่น
- [✅] แจ้งเตือน + แสดงป้าย Warning บน Profile ให้คนอื่นเห็น
- [ ] แจ้งเตือน + ระงับบัญชีชั่วคราว (เช่น 7 วัน ใช้งานไม่ได้)
- [ ] อื่นๆ: ___

## 2. Chat — ส่งรูปภาพได้ไหม?

ระบบมี Chat ทั้งใน Match (1:1) และ Sport Session / Event (Group)

**Q:** ใน Chat ผู้ใช้ส่งอะไรได้บ้าง?
- [ ] ข้อความ (Text) เท่านั้น
- [✅] ข้อความ + รูปภาพ + ยกเลิกข้อความ

## 3. Event Check-in — ทำงานอย่างไร?

questions2.md มีตัวอย่าง Event ที่มีขั้นตอน Check-in  
แต่ยังไม่รู้ว่า Check-in ทำงานอย่างไรในระบบ

**Q:** ระบบ Check-in ของ Event ทำงานอย่างไร?
- [✅] Host / Organizer กดยืนยันผู้เข้าร่วมเองในระบบ โดยให้นิสิตเปิด qrcode ของนิสิต ให้  Host / Organizer  เข้าใน event นั้น แล้วเปิดกล้องสแกน qrcode ของนิสิต qrcode ของนิสิตของใครของมัน
- [ ] ผู้เข้าร่วมกดปุ่ม "Check-in" เองเมื่อถึงงาน
- [ ] ไม่มีระบบ Check-in ใน MVP
