## 2. Sport Invite — ฟอร์มที่ส่งมีข้อมูลอะไรบ้าง?

Q9.2 บอกว่า เมื่ออีกคนตอบรับ → ระบบสร้าง Public Session อัตโนมัติ  
แต่ยังไม่รู้ว่าตอนส่ง Invite ต้องกรอกอะไร

**Q:** ตอนส่ง Sport Invite ผู้ส่งต้องกรอกข้อมูลอะไรบ้าง?

| ข้อมูล | ต้องกรอก | ไม่ต้อง |
|---|---|---|
| กีฬา |✅| |
| วันที่ |✅| |
| เวลา |✅ปรับเปลี่ยนได้| |
| สถานที่ |✅ปรับเปลี่ยนได้| |
| จำนวนผู้เล่นที่ต้องการ (max_players) |✅ปรับเปลี่ยนได้| |
| ข้อความเชิญ (optional message) |✅ปรับเปลี่ยนได้,กรอกไม่กรอกก็ได้ | |
